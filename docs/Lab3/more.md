# Lab3

1. 关于乘除法，有以下几种难度可以递进选择，大家根据自己的兴趣及个人能力选择，当然也可以实现多种。以 ***有符号、无溢出判断*** 为推荐的标准要求。

   - 无符号、无溢出判断
   - 无符号、有溢出判断
   - 有符号、无溢出判断
   - 有符号、有溢出判断

   对于实现*有符号*版本的同学，可以通过判断输入正负性先得到结果的正负性，后转化为无符号乘法，最后调整结果到合适的正负性；也可以查看 Booth's Algorithm 并实现。

   - 简单的舍入处理
